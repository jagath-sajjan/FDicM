# FDicM

Blazing fast responsive split pane terminal dictionary app engine built with Go and Bubble Tea

## Global Controls

TAB      Toggle focus panel viewport frames natively
p        Stream live audio pronunciation vocal sample elements via text to speech
v        Toggle raw JSON payload interface structure visibility data trees on the fly
c        Snapshot capture full clean structural copy definition out to system clipboard
q        Escape current result window frame or terminate application process cleanly
ESC      Escape current result window frame or terminate application process cleanly
